
# TASK MANAGER API

These are APIs for news Aggregator


## Run Locally

Clone the project

```bash
  git clone https://github.com/Maianki/news-aggregator-api.git
```

Go to the project directory

```bash
  cd news-aggregator-api 
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon app.js
```
